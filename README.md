# Design and Analysis of Algorithms (DAA) Lab Programs

## Overview
This repository contains the implementation of various Design and Analysis of Algorithms (DAA) laboratory programs developed using Python. The programs demonstrate fundamental algorithmic concepts including searching, string matching, graph algorithms, and shortest path computation. Each implementation is designed to improve understanding of algorithm design, efficiency, and real-world applications.

## Programs Included

### 1. Implementation and Performance Analysis of Interpolation Search
- Implements the Interpolation Search algorithm for searching elements in a sorted array.
- Analyzes the performance of the algorithm.
- Demonstrates its efficiency on uniformly distributed data.

### 2. Comparative Analysis of Naive, Rabin-Karp, and KMP Algorithms for String Matching
- Implements three popular string matching algorithms:
  - Naive String Matching
  - Rabin-Karp Algorithm
  - Knuth-Morris-Pratt (KMP) Algorithm
- Compares their performance and efficiency.

### 3. Implementation of Kruskal's and Prim's Algorithms for Minimum Spanning Tree
- Implements Kruskal's Algorithm using Union-Find.
- Implements Prim's Algorithm using a Min-Heap (Priority Queue).
- Finds the Minimum Spanning Tree (MST) and calculates its total cost.

### 4. Implementation of Single Source Shortest Path Algorithm (Dijkstra's)
- Implements Dijkstra's Algorithm using a Min-Heap.
- Finds the shortest path from a source vertex to all other vertices.
- Displays both the shortest distances and corresponding paths.

## Time Complexity

| Algorithm | Time Complexity |
|-----------|-----------------|
| Interpolation Search | Best: O(1), Average: O(log log n), Worst: O(n) |
| Naive String Matching | O(nm) |
| Rabin-Karp | Average: O(n + m), Worst: O(nm) |
| KMP | O(n + m) |
| Kruskal's Algorithm | O(E log E) |
| Prim's Algorithm | O((V + E) log V) |
| Dijkstra's Algorithm | O((V + E) log V) |
